
# AR_MacChanger

Kali Linux için özelleştirilmiş bir mac changer tool'u.


## Kullanım

Programı çalıştırmak için aşağıdaki komutu çalıştırın

```bash
  python ar_mac_changer.py -i eth0 -m 00:11:22:33:44:66

```
